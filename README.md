🛒 E-Commerce Web Application

This project is a simple full-stack E-commerce web application built using React and Node.js.
It allows users to browse products, search items, add them to cart, and view total price similar to a mini Amazon shopping experience.

🚀 Features
🛍️ View multiple products with images
🔍 Search products by name
🛒 Add items to cart
➕ Increase quantity automatically
💾 Cart stored using localStorage (data persists on refresh)
👤 Basic login using username
💰 Total price calculation
🎨 Clean and responsive user interface

🛠 Tech Stack
Frontend: React.js
Backend: Node.js, Express.js
Styling: CSS

▶️ How to Run

Backend
cd server
npm install
node index.js

Frontend
cd client
npm install
npm start
